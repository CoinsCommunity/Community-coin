# Formal Protocol Specification

This directory contains the formal specification of the Community protocol.

The specification is written before any code exists. It defines every rule, every parameter, every cryptographic construction precisely enough that an independent team could implement Community from scratch using only this document.

**Status: In progress**

Sections will be added as they are completed and reviewed.

# Community (CMTY)

> *They never built any of this for us. Not this time.*

**Community is a peer-to-peer electronic currency for the entire world.**

Not for the wealthy. Not for the connected. Not for the people who already have access.

For everyone the system forgot, excluded, surveilled, or punished.

---

## What Makes Community Different

| Property | Bitcoin | Community |
|---|---|---|
| Privacy | None by default | Mandatory — sender, receiver, amount all hidden |
| Quantum resistant | No | Yes — NIST 2024 standards built in from day one |
| Mining | ASIC dominated | CPU only — any laptop or phone can mine |
| Consensus | Proof of Work | Hybrid PoW + PoS — two independent security layers |
| Supply | Hard cap — miners eventually unpaid | Tail emission — miners incentivized forever |
| Governance | Informal developer control | None — immutable on deploy |
| Premine | No | No |
| Company | No | No |
| Kill switch | No | No |

## The Mission

The financial system has been captured and recaptured for ten thousand years.

Grain became taxes. Gold became confiscation. Bitcoin became BlackRock.

Central Bank Digital Currencies are being deployed by 130+ countries right now. China's social credit system already freezes citizens out of the economy based on behavior scores. The next generation of financial control won't come from banks — it will come from AI systems owned by three companies deciding your credit score, your loan eligibility, and your access to housing.

Community is the answer to all of it. Not a better product. Not a reformed system. A replacement — autonomous, private, ungovernable, and built for everyone simultaneously.

## Philosophy

This protocol will be deployed once and never controlled again.

No admin access. No upgrade mechanism. No kill switch. No governance.

What ships is what runs. Forever.

That demands it be built correctly the first time — which is why the development process is staged, public, and takes as long as it needs to take.

Security through overwhelming public scrutiny. Not security through trust.

---

*Community is not a company. It is not a product. It has no owner, no headquarters, no legal entity, no employees, and no administrator. It is a protocol, released to the world, belonging to everyone who uses it and no one who built it.*
